Project Build Guide

🕹️ Controls & Hotkeys

• R-Click + Move: Look Around / Fly View
• WASD: Fly Movement (when looking)
• M-Click / Space + Drag: Pan Camera
• Alt + M-Click: Orbit Mode (Rotate around target)
• Scroll Wheel: Zoom In / Out
• F Focus: Frame project in view
• G Grid: Toggle Snap to Grid
• Ctrl + Z Undo: Step back
• Ctrl + Y Redo: Step forward

🔧 Tool Usage

• Select: Click any part to view its data in the Inspector.
• Draw Tube: Click a node to start, then click another to build a tube.
• Point Edit: Drag nodes or mount points to adjust geometry.
• Link / Group: Select multiple parts, then click "Confirm Link" to group them.
• Join Nodes: Drag one node onto another to merge them permanently.
• Sculpt Body: Select purple body nodes and drag them to skin the chassis.

📚 Building Workflow

1. Foundation: Place Suspension and Hubs to define your wheelbase.
2. Structure: Connect Hubs and Mounts using the Draw Tube tool. Use I-Beams for heavy frames.
3. Mechanicals: Add your Engine and Gearbox. Use the Transfer Case for 4x4 off-roaders.
4. Plumbing: Route Fuel and Brake lines between components.
5. Finishing: Sculpt the body panels, add lights, and click "Calculate Build" for a full report.
